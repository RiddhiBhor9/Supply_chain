# Supply_chain
📦 Supply Chain Data Analysis

A data analysis project focused on exploring, cleaning, and visualizing supply chain data using Python. This project demonstrates key data analysis workflows including preprocessing, exploratory data analysis (EDA), and visualization.

🚀 Project Overview

This project analyzes a supply chain dataset to extract meaningful insights such as:

Product performance
Sales distribution
Data quality handling (missing values, duplicates)
Basic feature engineering
🛠️ Tech Stack
Python 🐍
Pandas
NumPy
Matplotlib
Jupyter Notebook / VS Code
📂 Project Structure

🔍 Features
✅ Data loading and inspection
✅ Data cleaning (handling missing values, duplicates)
✅ Column standardization
✅ Exploratory Data Analysis (EDA)
✅ Data visualization
✅ Exporting cleaned data
📊 Sample Analysis Performed
Distribution of product types
Number of products sold
Summary statistics of numerical features
Data consistency checks
⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/RiddhiBhor9
Navigate to the project folder:
cd supply-chain-analysis
Install dependencies:
pip install pandas numpy matplotlib
Run the script:
python supply_chain.py
